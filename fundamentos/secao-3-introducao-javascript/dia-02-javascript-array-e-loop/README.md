# Módulo 1 - Dia 3.2: JavaScript - Array e loop For

> Data: 29/03/2023

Hoje abordamos de forma mais profunda sobre as estruturas de repetição (for, for/of, while e do/while) e sobre a estrutura de dados "lista" (mais conhecida como "array"). Além do conteúdo teórico, recebemos diversos insights sobre Lógica de Programação, o que são Algoritmos e diversos exercícios para praticar lógica de programação com arrays e estruturas de repetição. Vale destacar a abordagem sobre o Algoritmo de Ordenação Bubble Sort durante os exercícios, onde nos foi apresentado a teoria que fundamenta seu funcionamento e passados exercícios para produzirmos manualmente o algoritmo Bubble Sort, objetivando organizar um array de números em ordem crescente e decrescente.
